LifeSchedulerBot - это телеграм бот, предназначенный для огранизации жизни пользователя. 
Нашему боту, в отличие от записной книжки, мы делегируем не только задачу хранения информации, но и задачу составления расписания.
Создавая данного бота, мы пользуемся следующими предположениями:
    1) бот сможет найти более подходящее время для дела, чем пользователь
    2) бот сможет быстрее пользователя найти подходящее время для дела
    3) пользователь дает достаточно информации о деле, для грамотного планирования

Основные задачи, которые решает наш бот:
    1) запись событий, фиксированных во времени
    2) запись событий без фиксированного времени на какое-то время, с помощью алгоритма планирования
    3) запись повторяющихся событий
    4) вывод расписания на указанный день и на день недели
    5) вывод статистики по дням (суммарные физические и умственные энергозатраты)
